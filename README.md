# Todo App

A modern and interactive Todo App built with React that helps you manage daily tasks efficiently with features like filtering, editing, and persistent storage.

> [Live demo](https://todo-ayze8wk9m-sujan-bks-projects.vercel.app/)

---

## Built With

- React (Vite)
- Tailwind CSS v4
- React Icons
- LocalStorage API

---

## Features

- Add new tasks
- Delete tasks
- Toggle task completion
- Edit tasks
- Filter tasks (All / Active / Completed)
- Persistent storage using localStorage
- Fast and responsive UI

---

## Installation & Setup

Follow these steps to run the project locally:

### 1. Clone the repository

```bash
git clone https://github.com/sujan0017/todo-app.git
cd todo-app
```

### 2. Install dependencies

```bash
npm install
```

### 2. Run the dev Server

```bash
npm run dev
```
